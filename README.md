# Te Ara

Te Ara is a typeface extrapolated out from a logotype developed by the font author for the Te Ara o Mauao Gallery at Toi Ohomai's campus, here in Tauranga New Zealand.

The original characters were "M, A, U, L O, G, R, Y, E", enough to get a typefaces DNA from.
